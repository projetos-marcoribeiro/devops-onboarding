## Portais e Ferramentas

### Service Desk
🔗 [https://hotmart.atlassian.net/servicedesk](https://hotmart.atlassian.net/servicedesk)

Portal oficial para solicitação de acessos, abertura de tickets de suporte e acompanhamento de chamados. É por aqui que passam todas as solicitações de acesso a sistemas e ferramentas da empresa.

---

### Hotmart AI
🔗 [https://chat.hotmart.ai](https://chat.hotmart.ai)

Assistente de IA interno da Hotmart. Pode ser usado para tirar dúvidas, rascunhar documentos, explorar conceitos técnicos e auxiliar em tarefas do dia a dia. Prefira este ao ChatGPT para garantir que informações sensíveis da empresa não saiam do ambiente corporativo.

---

### Google Chat Bot
🔗 [https://hotmart.atlassian.net/wiki/spaces/ddocs/pages/1372782650/Google+Chat+Bot+Crie+chatbots+no+Google+Chat](https://hotmart.atlassian.net/wiki/spaces/ddocs/pages/1372782650/Google+Chat+Bot+Crie+chatbots+no+Google+Chat)

Documentação sobre como criar e usar chatbots no Google Chat. Útil para automações de notificações, alertas de incidentes e integrações com ferramentas internas via chat.

## Repositórios GitHub

### DevOps Docs
🔗 [https://github.com/Hotmart-Org/devops-docs](https://github.com/Hotmart-Org/devops-docs)

Repositório central de documentação do time DevOps. Contém guias, runbooks, arquitetura da plataforma e referências técnicas. É a fonte da verdade para documentação operacional.

---

### GitHub Actions: Workflows Reutilizáveis
🔗 [https://github.com/Hotmart-Org/actions](https://github.com/Hotmart-Org/actions)

Repositório com GitHub Actions reutilizáveis (reusable workflows) mantidos pelo time DevOps. Os times de produto usam esses workflows nos seus pipelines de CI/CD. Se você vai trabalhar com pipelines, este repositório é leitura obrigatória.

---

### Helm Infrastructure
🔗 [https://github.com/Hotmart-Org/devops-helm-iac](https://github.com/Hotmart-Org/devops-helm-iac)

Repositório com os Helm charts de infraestrutura usados para deploy de aplicações nos clusters EKS. Contém os charts base e configurações de infraestrutura gerenciadas pelo time DevOps.

---

### Terraform Base Module
🔗 [https://github.com/Hotmart-Org/terraform-base-module](https://github.com/Hotmart-Org/terraform-base-module)

Módulo Terraform base da Hotmart. É o ponto de partida para provisionamento de infraestrutura na AWS. Todos os times que criam recursos AWS devem usar este módulo como base para garantir padronização, segurança e conformidade com as políticas da empresa.
